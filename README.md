# 🎓 Платформа самообучения (Django + DRF)
### REST API для платформы онлайн-образования. Проект позволяет управлять пользователями, курсами, материалами и тестами с разграничением прав доступа.

## 🛠 Технологии
* Backend: Python 3.12, Django 5.x, Django Rest Framework (DRF).
* База данных: PostgreSQL.
* Аутентификация: JWT (JSON Web Tokens).
* Документация API: Swagger (drf-yasg).
* Управление зависимостями: Poetry.
## 🚀 Быстрый старт
#### Клонируйте репозиторий
    git clone https://github.com/AleksChek26/DRFProject
#### Установите зависимости
    poetry install
#### Создайте базу данных
#### Настройте переменные окружения
    Скопируйте пример файла .env.example в .env и заполните данные.
#### Примените миграции базы данных
    python manage.py migrate
#### Создайте суперпользователя (администратора)
    python manage.py createsuperuser
#### Запустите сервер разработки
    python manage.py runserver
## 🌐 Доступ к API и Админке
* Админ-панель Django: http://127.0.0.1:8000/admin/
* Документация API (Swagger): http://127.0.0.1:8000/api/docs/
* Красивая документация (ReDoc): http://127.0.0.1:8000/api/redoc/
## 🧪 Тестирование
#### Проект покрыт тестами.
## ✅ Роли и права доступа
#### Система поддерживает три роли:
* Администратор: Полный доступ ко всему, включая админ-панель.
* Преподаватель: Может создавать свои курсы, материалы и тесты.
* Студент: Может только просматривать материалы и проходить тесты.
#### Примечание: Преподаватель видит и управляет только теми объектами, владельцем которых он является.
